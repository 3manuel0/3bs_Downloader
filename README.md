# Installing:

**⚠️ WARNING:**
Project, still not finished yet.
you need to have `python3` and `ffmpeg` on your system.

change your path or open terminal on the script's folder then you create a virtual environment:

for windows:

```shell
    python -m venv .venv
```

for linux:

```bash
    python3 -m venv .venv
```

then run:

for windows

```shell
    .venv\Scripts\activate.bat
```

for linux (bash/zsh):

```bash
    source .venv/bin/activate
```

now to install requirments:

```bash
    pip install -r req.txt
```

# Runing:

make sure you're always using venv or maybe you can use other methodes like `Conda` then:

```bash
    python main.py
```

you will be prompted to enter the link do it and enjoy.
